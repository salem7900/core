### Description

The goal of the project is to code a C library that will include numerous general purpose
functions for our programs.

The project includes: 
- 23 standard Libc functions;
- 11 additional functions;
- 9 bonus linked list functions;
- libft.h library;
- Makefile with the following functions: NAME, all, clean, fclean, re.

### Instructions

To compile the mandatory part of the project, run make. This will generate the static library libft.a.

To compile the bonus part, use make bonus.

To remove the object files, run make clean. To remove both the object files and the generated library, run make fclean. If you want to rebuild everything from scratch, use make re.

To use the library in another project, include the header file libft.h in your source code and link the library when compiling your program.
